# 🎮 UniRetro

**UniRetro** is a bird's-eye view 2D digital recreation of the
Azerbaijan Architecture and Construction University campus built using
GameMaker Studio. The project is mainly a learning experience focused on
level design, top-down movement systems, and recreating a real-world
environment in a simple interactive format.

This is not a fully realistic simulation --- it's more like a playable
digital campus map made by a student developer learning game development
step by step.

## 🎯 Goals of the Project

### Primary Goals

-   Recreate the campus layout in 2D.
-   Implement basic player movement.
-   Learn GameMaker Studio fundamentals.
-   Practice object and room management.

### Secondary Goals

-   Understand camera systems.
-   Learn collision handling.
-   Improve tilemap design skills.
-   Build a structured game project from scratch.

## 🎮 Controls

W / ↑ : Move Up S / ↓ : Move Down A / ← : Move Left D / → : Move Right

------------------------------------------------------------------------

## ⚙️ Technical Details

### Player System

-   Directional movement
-   Speed control
-   Collision checking
-   Sprite direction switching

### Collision System

-   Rectangle collision masks
-   Solid building objects
-   Simple physics

### Camera System

-   Player-follow camera
-   Room boundary restriction
-   Planned smooth camera interpolation

## ⚠️ Known Issues / Limitations

-   Some buildings are placeholders.
-   Interiors are mostly missing.
-   Collision may feel slightly rough.
-   Art style is inconsistent.
-   Performance optimization is minimal.
